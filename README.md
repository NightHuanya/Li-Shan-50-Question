      所謂二分逼近法，解釋如下：
      設某數x位於線段A。B。=[a。﹐b。]內，(亦即a。≦x≦b。)，vhbdi vfhae
      hjhneA。B。等分為兩段，亦即[a,(a。+ b。)/2]和[(a。+ b。)/2,b。]；而把x
      所在的那一段稱為A1B1=[a1,b1](如果x=(a。+ b。)/2,則任取其中一段)，換句話
      說，當a。≦x≦(a。+ b。)/2時，a1=a。，b1=(a。+ b。)/2，當(a。+ b。)/2≦
      x≦b。時，a1=(a0+b0)/2，b1=b。，這樣逐次二等分而縮小範圍，就可得到x，
      此法為二分逼近法。
